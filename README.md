# Holberton School - Higher Level Programming

This repository contains projects for the Holberton School "Higher Level Programming" curriculum. It is organized as a collection of Python practice tasks and example scripts.

## Repository Structure

- `python-hello_world/`
  - `2-print.py` - Example Python script demonstrating printing and string formatting.

## Getting Started

### Prerequisites

- Python 3.x installed on your system.
- A code editor or terminal for running Python scripts.

### Running the Example

From the repository root, run:

```bash
python3 python-hello_world/2-print.py
```

## Notes

- This repository currently contains a small subset of tasks from the curriculum.
- Add new folders and scripts under the repository root as you continue through Holberton School projects.
