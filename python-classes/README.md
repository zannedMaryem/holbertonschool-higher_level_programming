# Python - Classes and Objects

## Description

This project covers the fundamentals of Object-Oriented Programming (OOP) in Python. You will learn how to create classes, instantiate objects, and use various features of Python's class system including attributes, methods, encapsulation, and properties.

## Learning Objectives

- **What is OOP** - Understanding the paradigm of Object-Oriented Programming
- **"First-class everything"** - How Python treats functions and classes as first-class objects
- **What is a class** - How to define and structure classes in Python
- **What is an object and an instance** - The difference between class definitions and object instances
- **What is the difference between a class and an object or instance** - Understanding the class as a blueprint and instances as concrete objects
- **What is an attribute** - Data stored within a class or instance
- **What are and how to use public, protected and private attributes** - Using naming conventions to control attribute visibility
- **What is self** - Understanding the `self` parameter in instance methods
- **What is a method** - Functions defined within a class that operate on instances
- **What is the special `__init__` method and how to use it** - The constructor method for initializing new instances
- **What is Data Abstraction, Data Encapsulation, and Information Hiding** - Principles of protecting internal object state
- **What is a property** - Using the `@property` decorator to create managed attributes
- **What is the difference between an attribute and a property in Python** - Direct data access versus computed/managed properties
- **What is the Pythonic way to write getters and setters in Python** - Using `@property` and `@setter` decorators
- **How to dynamically create arbitrary new attributes for existing instances of a class** - Adding attributes to objects at runtime
- **How to bind attributes to objects and classes** - Understanding instance vs class attributes
- **What is the `__dict__` of a class and/or instance of a class and what does it contain** - Viewing the namespace of classes and instances
- **How does Python find the attributes of an object or class** - Understanding the attribute lookup process and MRO
- **How to use the getattr function** - Dynamic attribute access using built-in functions
