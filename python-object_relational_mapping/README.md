# Python - Object Relational Mapping

This project is designed to help learn the fundamentals of Object-Relational Mapping (ORM) and SQLAlchemy using Python.

## Objective

The main goal is to understand how Python applications can interact with relational databases without writing raw SQL for every query. This repository focuses on:

- connecting Python to a MySQL database
- using SQLAlchemy for database operations
- mapping Python classes to database tables
- performing CRUD operations through ORM concepts
- understanding the relationship between Python objects and SQL databases

## Topics Covered

- ORM concepts and principles
- SQLAlchemy core and ORM usage
- database models and sessions
- query execution and filtering
- relationships between tables
- safe and efficient database access

## Learning Goals

By working through this project, you will be able to:

- explain what ORM is and why it is useful
- create and manage database models with SQLAlchemy
- perform database queries using Python objects
- build simple applications that interact with databases through ORM

## Notes

This repository is a practical introduction to working with databases in Python using SQLAlchemy and ORM patterns.
